# GameOver Financies — SaaS Financial Dashboard 🎮🔨

Sistema web premium para controle financeiro pessoal, planejamento de metas e organização visual estratégica.

## 🚀 Tecnologias

- **Frontend**: Next.js 14 + TypeScript + Vanilla CSS (Design System customizado)
- **Backend**: Node.js + Express + Prisma
- **Banco**: PostgreSQL (Produção via Neon.tech)
- **Auth**: JWT em cookies httpOnly + bcrypt (Seguro contra XSS)
- **Email**: Resend API (Transactional emails)

## 📦 Estrutura do Projeto

```
gameover-financies/
├── apps/
│   ├── backend/     # Express API
│   └── frontend/    # Next.js App
├── docker-compose.yml
└── README.md
```
